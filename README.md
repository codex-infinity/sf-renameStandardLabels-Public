# Rename Standard Labels Utility Item

## Description

The **Rename Standard Labels Utility Item** tool is a Salesforce solution that addresses a known issue where standard Utility Bar item labels (such as "History", "Recent Items", etc.) are not translated according to the user's language preference.

This solution enables you to:

## How to Use

1. Install the Unlocked package using the link above
2. Go to Setup > Lightning App Builder
3. Open or create a Lightning App
4. Configure the Utility Bar by adding the `renameStandardLabelsInUtility` component
5. Labels will be automatically translated based on the user's language
